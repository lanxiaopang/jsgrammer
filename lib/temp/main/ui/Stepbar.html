Stepbar步骤条控件

步骤条用于在多步操作中向用户指示当前进度。步骤条也被称为进度条（progress bar）但为了和显示百分比进度的进度条进行区分，这里称为步骤条（step bar）。

步骤条通常由若干个圆形节点和水平的条形背景形成的单元所组成。每个节点可以有“未激活”、“已激活”和“以完成”三种显示状态。

控件调用方法

            require(['stepbar'], function () {
                var stepbar=$("selector").stepbar(...);
                stepbar.next();
                stepbar.prev();
                stepbar.active(index,"runing/complete");
            });

参数说明

参数名 作用 类型 默认值 备注
data 步骤数据 Array 必须设置
value 默认激活哪一项 Int 0 从0开始
theme 主题样式 String stepbar-default
change 当步骤反生改变时执行 Function
complete 当完成所有步骤时执行 Function

接口说明

next

激活下一步，若没有下一步则表明全部完成

            require(['stepbar'], function () {
                var stepbar=$("selector").stepbar(...);
                stepbar.next();
            });

prev

激活上一步，若没有上一步则不执行

            require(['stepbar'], function () {
                var stepbar=$("selector").stepbar(...);
                stepbar.prev();
            });

active

使某一步变为正在执行或已完成，需要传递两个参数，第一个为步骤索引从0开始，第二个为状态runing/complete，若不填则默认为runing

            require(['stepbar'], function () {
                var stepbar=$("selector").stepbar(...);
                stepbar.active(index,"runing/complete");
            });

参考实例

上一步/下一步

    

    
        var step;
        function scriptComplete() {
            require(["stepbar"], function () {
                step = $("#demo").stepbar({
                    data: ["填写账户信息", "绑定银行卡", "设置电子账户交易密码", "开户成功"],
                    complete: function () {
                        alert("全部完成");
                    }
                });
            });
        }
        function prev() {
            step.prev();
        }
        function next() {
            step.next();
        }

指定某一项的状态

    

    
        var  step;
        function scriptComplete() {
            require(["stepbar"], function () {
                step = $("#demo").stepbar({
                    data: ["填写账户信息", "绑定银行卡", "设置电子账户交易密码", "开户成功"],
                    complete: function () {
                        alert("全部完成");
                    }
                });
            });
        }
        function active() {
            var stepIndex = $("#steps").val();
            var tag = $("#tag").val();
            step.active(stepIndex, tag);
        }
