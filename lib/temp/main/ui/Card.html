Card-卡片布局控件

Card卡片布局控件，此控件用于控制多个容器的显示和隐藏。

控件调用方法

            require(['card'], function (c) {
                //stack 即 stack 对象
                var card=new s(...);
                card.show("name");
                card.show(0);
                card.next();
                card.prev();
            });

参数说明

Array 类型

jQuery Selector/jQuery Element

例如：["#id",".className",$("selector"),"[data-name='name']"]

JSON 类型

采用别名标识的方式，例如：

     {
        stepOne:"#id",
        stepTow:".className",
        otherStep:$("selector")
     }

接口说明

show

支持两种形式的参数：String / Int； String 时代表值为JSON中的name值，Int 时代表索引，若在初始化时，传入的是JSON参数，则String和Int 都可以；若为Array时，只能使用Int 索引进行操作。

            require(['card'], function (c) {
                //stack 即 stack 对象
                var card=new c(...);
                card.show("name");
                card.show(0);
            });

next

显示下一个

            require(['card'], function (c) {
                //stack 即 stack 对象
                var card=new c(...);
                card.next();
            });

prev

显示上一个

            require(['card'], function (c) {
                //stack 即 stack 对象
                var card=new c(...);
                card.prev();
            });
