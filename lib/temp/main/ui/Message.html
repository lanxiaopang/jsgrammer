Message-消息提醒

消息提醒控件，用于友好的提示给用过操作结果或提示消息。

控件调用方法

            require(['message'], function (m) {
                //m 即 message 对象
                //m.open(..);
                //m.warn(..)
                //...
            });

接口说明

open

打开一个消息提醒，通过传入不通的配置信息，展示不同的消息提醒

参数说明

参数名 作用 类型 默认值 备注
content 消息内容 String
click 点击执行方法 Function
timer 多少秒后消失 秒 10 若timer为0，则表示不自动消失
theme 主题样式 String default 可选值：warn、error、success、info、default，也可以自定义样式
position 显示位置 Array ["right","top"] 右上角 注意：第一个为水平方位（x），第二个为垂直方位（y）,类似于jquery-ui中的position方法。

暂时支持：["left","top"]、["left","bottom"]、["right","top"]、["right","bottom"] 4种方位。

参考实例-（全参数）

           require(["message"], function (m) {
                m.open({
                    content: "有人给您发送了一条消息",
                    theme: "default",
                    timer: 10,
                    click: function () {
                        alert("您点击了一条消息");
                    }
                });
            });

参考实例-（简单参数数）

           require(["message"], function (m) {
                m.open("提示消息",function(){
                    //click
                });
            });
