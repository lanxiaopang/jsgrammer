Progressbar-进度条控件

进度条用于显示一个百分比进度，由一个外框和一个表示进度的内框构成，另外还有一个显示文字的标签位于进度条内部或外侧上下左右四个方向上。

控件调用方法

            require(['progressbar'], function () {
                $("selector").progressbar(...);
            });

参数说明

参数名 作用 类型 默认值 备注
max 最大值 Int 100
theme 主题样式 String progress-default
value 默认值 Folat 0
tipText 自定义tip文本显示内容 Function 0% 调用此回调时会将当前百分比传入，并要求返回显示内容。与showTip相关联。
showTip 是否显示气泡提示 Bool true
change 发生改变时触发事件 Function
complete 当到达最大值时触发事件 Function

接口说明

setValue

设置进度条值，必须大于等于0，并且小于等于最大值max

    
            require(["progressbar","jqueryui"], function () {
                var progressbar= $("#demo").progressbar();
                progressbar.setValue(20);
            });

参考实例

    
    

    
    
            require(["progressbar","jqueryui"], function () {
                var progressbar= $("#demo").progressbar({value:20});
