Loading-等待控件

此空间可以显示Loading界面，可用于页面加载等情况，是页面更友好。

控件调用方法

            require(['loading'], function (l) {
                //loading 即 loading 对象
                var loading=new l(...);
                //loading.open();
                //loading.close();
            });

参数说明

参数名 作用 类型 默认值 备注
to 需要输出到的容器 Selector/jQuery Element 注意：必须设置此参数。
className 样式名称 String loading-default 默认提供：loading-default和loading-point两种。
text 显示文本 String 请耐心等待...

接口说明

open

打开Loading等待界面

            require(['loading'], function (l) {
                //loading 即 loading 对象
                var loading=new l(...);
                loading.open();
            });

close

关闭Loading等待界面

            require(['loading'], function (l) {
                //loading 即 loading 对象
                var loading=new l(...);
                loading.close();
            });

参考实例

       function demo() {
            require(["loading"], function (t) {
                var loading = new t({
                    to: "body",
                    text: "3秒后自动消失"
                });
                loading.open();
                setTimeout(function () {
                    loading.close();
                }, 3000);
            });
        }
        function demo1() {
            require(["loading"], function (t) {
                var loading = new t({
                    to: "body",
                    className:"loading-point",
                    text: "3秒后自动消失"
                });
                loading.open();
                setTimeout(function () {
                    loading.close();
                }, 3000);
            });
        }
