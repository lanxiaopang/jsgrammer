Tab-标签页控件

tab标签页控件可实现标签页之间的互相切换

控件调用方法

            require(['tab'], function (t) {
                //tab 即 tab 对象
                var tab=new t(...);
            });

参数说明

参数名 作用 类型 默认值 备注
to 需要输出到的容器 Selector/jQuery Element 注意：必须设置此参数。
defaultSelect 默认选中tab TabName/TabIndex 0 可以设置为Tab的name值，或者Tab的index索引。默认为0
onselect 当标签被选中后执行事件 Function 若重复选中同一个tab标签，只会执行一次此事件。
onfocus 当标签页选中时执行事件 Function 若重复选中同一个tab标签，会执行多次此事件。执行顺序是先onselect然后onfocus。
onblur 当标签页失去焦点时执行事件 Function 若方法返回false时，则会终止tab页的切换。可用作对tab内容验证。
maxTabLength 最大tab项个数 Int 10 此方法只在add动态添加tab项中起作用，若页面HTML中已存在超过限制的个数时不会提示。默认最大10个。
className 样式名称 String tab-default

接口说明

selectTab

选中一个标签页，参数可以为标签页的name也可是tabIndex索引(从0开始)。

            require(['tab'], function (t) {
                //tab 即 tab 对象
                var tab=new t(...);
                tab.selectTab("name");
                tab.selectTab(0);
            });

add

动态添加一个标签页

参数说明

参数名 作用 类型 默认值 备注
name tab标签name值
text tab显示文本 name和text至少要设置一个
content tab标签页内容 String/HTML
className 自定义标签样式 String onselect然后onfocus。
autoSelect 是否默认选中 Bool true
hideClose 是否隐藏关闭按钮 bool false

调用实例

           require(['tab'], function (t) {
                //tab 即 tab 对象
                var tab=new t(...);
                tab.add(...);
            });

remove

移除指定的tab标签，参数可以是tabName或者tabIndex。

           require(['tab'], function (t) {
                //tab 即 tab 对象
                var tab=new t(...);
                tab.remove("name");
                tab.remove(0);
            });
