ToolTip-提示气泡控件

提示气泡主要用于用户鼠标悬浮于某一元素之上时弹出一个小的消息气泡向用户显示提示信息。

提示气泡也可以用于主动悬浮在一个元素的top、bottom、left、right四个方位。

控件调用方法

标签式调用方式

可以直接在DOM元素中添加指定的属性，来设置提示气泡的信息。但页面中必须引用tootip组件。

此方法适用于所有类型的标签，其中元素必须设置role为tooltip，以表明要采用ToolTip控件展示，

data-tooltip-position为展示位置，默认为top；data-tooltip-class为自定义tooltip样式，默认为tooltip-default。

脚本式调用方式

            require(['tooltip'], function (t) {
                //t 即 tooltip 对象
                //t.show(..);
                //t.hide(..);
            });

接口说明

show

可以主动显示role="tooltip"元素的提示信息，也可以为没有设置role的元素展示一个提示气泡。

参数说明

若传入jQuery Element/Selector可以显示已经设置role="tooltip"的提示。

若要重新展示一个气泡提示，请参考下面的配置参数：

参数名 作用 类型 默认值 备注
of 需要展示的控件 Selector/jQuery Element 必须设置此参数
to 气泡提示输出到哪个容器内 Selector body
content 提示内容 String
className 自定义样式 String tooltip-default
maxWidth 最大宽度 Int 250
width 宽度 String auto
textalign 文本靠齐类型 String left
position 显示位置 String top 可选值：top、bottom、left、right
fit 当内容尺寸超出范围，是否允许自动调整 bool false 默认值为false，若鼠标移动触发的气泡提示则fit总是为true

此属性为高级应用，在表单验证或其他场合fit会解决部分超出定位问题。

参考实例

            require(['tooltip'], function (t) {
                //t 即 tooltip 对象
                t.show("#demo");
                t.show("#demo","提示内容");
                t.show("#demo","提示内容","bottom","className");
                t.show({
                    of:"#demo",
                    content:"提示内容",
                    className:"className",
                    position:"left"
                });
            });

hide

隐藏元素的提示气泡，参数为jQuery Element/Selector

            require(['tooltip'], function (t) {
                //t 即 tooltip 对象
                t.hide("#demo");
            });

hideAll

隐藏指定容器下的所有气泡提示，参数为jQuery Element，默认是body

            require(['tooltip'], function (t) {
                //t 即 tooltip 对象
                t.hideAll();
                t.hideAll($("#container"));
            });

setText

设置已显示出来的tip中的值，参数为jQuery Element，第二个参数为value值

            require(['tooltip'], function (t) {
                //t 即 tooltip 对象
                t.setText("selector",value);
            });
