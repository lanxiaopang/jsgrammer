Overlay-遮罩

Overlay为遮罩层控件，是很多控件的基础功能，例如弹窗、Loading等，可以让我们限制用户禁止控制遮罩层下方的内容。

控件调用方法

            require(['overlay'], function (ov) {
                //实例化遮罩层控件
                var overlay = new ov();
            });

参数说明

参数名 作用 类型 默认值 备注
color 遮罩颜色 String #000
zindex 遮罩层z-index值 Int 500
to 遮罩层输出到的容器 Selector body 如果指定了to，需要为to设置position:relative
click 点击遮罩层触发事件 Function undefined

接口说明

open

打开遮罩层，当打开遮罩层时，若已经存在遮罩层，会隐藏掉其他遮罩层

            require(['overlay'], function (ov) {
                //实例化遮罩层控件
                var overlay = new ov();
                //打开遮罩层
                overlay.open();
            });

close

关闭遮罩层，若之前打开过多个遮罩层，当关闭遮罩层时，会自动显示出下一个遮罩层，参考showNext()接口

            require(['overlay'], function (ov) {
                //实例化遮罩层控件
                var overlay = new ov();
                //打开遮罩层
                overlay.open();
                //关闭遮罩层
                overlay.close();
            });

showNext

显示下一个遮罩层，不建议单独使用，在close时，会自动调用此方法

            require(['overlay'], function (ov) {
                //实例化遮罩层控件
                var overlay = new ov();
                //显示下一个遮罩层
                overlay.showNext();
            });
